1. 회로도대로 연결(저항은 빨강,주황,갈색,금색)
   ! 2번이 TX, 3번이 RX로 연결할 것! (0, 1번의 연결과 반대) !
2. 아두이노에 코드 입력 후 컴파일(18번째 줄에 와이파이 ID, Password 입력)
   (업로드 안될 시 esp8266 빼고 업로드 후 연결)
3. 메모장에 웹코드 입력 후 html로 저장(18번째 줄에 아두이노 ip 입력)
4. jquery.min.js 파일 html과 같은 경로상에 다운로드
5. html 클릭 후 LED버튼 클릭, 점등 확인
 ( 웹 작동시키는 기기와 아두이노 같은 와이파이 연결할 것)
